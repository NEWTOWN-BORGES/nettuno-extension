🕵️ NETTUNO — Política de Privacidade

Versão 1.0 · Última atualização: Maio de 2026 · Aplicável a partir da versão 4.0 da extensão
Resumo em linguagem simples (TL;DR):
Recolhemos apenas o mínimo necessário para identificar padrões de burla nos marketplaces. Nunca vendemos os teus dados. Nunca partilhamos o teu nome ou email com plataformas de terceiros. Os teus votos são usados apenas para estatísticas agregadas anónimas. Podes apagar a tua conta e todos os dados a qualquer momento.

1. Quem Somos

A extensão NETTUNO é desenvolvida por Neuton Borges (dev.nettuno@gmail.com), a título individual, com sede em Portugal. Para efeitos do RGPD, atuamos como Responsável pelo Tratamento dos dados descritos nesta política.

NÃO temos anúncios comportamentais. Os banners "PARCEIRO" são estáticos e não te rastreiam.

3. Como Usamos os Dados

  • Reputação comunitária: Agregamos votos de múltiplos utilizadores para mostrar um score de risco por anúncio. Os dados são sempre mostrados de forma agregada.
  • Relatórios estatísticos: Sinalizações verificadas podem ser incluídas em relatórios anónimos partilhados com autoridades (GNR, ASAE, Provedoria do Consumidor) e com plataformas como OLX/Vinted para combate a fraudes. O teu nome nunca consta nesses relatórios.
  • Melhoria do serviço: Dados agregados anónimos ajudam-nos a melhorar os algoritmos de deteção.

4. Com Quem Partilhamos

  • Google Firebase (Firestore + Auth): Plataforma de base de dados e autenticação. Política de privacidade Firebase. Processador de dados da UE com cláusulas contratuais padrão.
  • Autoridades competentes: Apenas mediante obrigação legal ou pedido formal com base legal.
  • Ninguém mais. Não vendemos, não alugamos, não cedemos dados a terceiros para fins comerciais.

5. Os Teus Direitos (RGPD)

  • Acesso: Podes ver todos os teus dados no painel de perfil da extensão.
  • Retificação: Podes editar alcunha, interesses e outros campos opcionais a qualquer momento.
  • Eliminação: "Apagar Conta e Dados" remove imediatamente o teu perfil do Firestore. Os votos ficam como dados anónimos (sem ligação ao teu UID).
  • Portabilidade: Contacta-nos por email para exportar os teus dados em JSON.
  • Oposição: Podes terminar sessão a qualquer momento — a extensão continua a funcionar em modo de leitura anónima.
  • Reclamação: Tens o direito de apresentar reclamação à CNPD (Portugal) ou à autoridade de proteção de dados do teu país.

6. Retenção de Dados

  • Perfil de utilizador: Mantido até apagares a conta.
  • Votos e sinais: Mantidos indefinidamente como dados estatísticos anónimos (sem ligação ao UID após eliminação de conta).
  • Sessão anónima: Dados de sessão Firebase anónima eliminados se não houver atividade em 12 meses.

7. Segurança

Os dados são armazenados no Google Firebase com encriptação em trânsito (TLS) e em repouso. O acesso ao Firestore é controlado por regras de segurança que garantem que cada utilizador só acede aos seus próprios dados. Votos são protegidos contra escrita por utilizadores não verificados.

8. Menores

A extensão não é direcionada a menores de 16 anos (ou 13 anos fora da UE). Não recolhemos intencionalmente dados de menores. Se tiveres conhecimento de dados de menores no sistema, contacta-nos imediatamente.

9. Conteúdo Patrocinado

Quando ativo, o banner "PARCEIRO" ou "PUBLICIDADE" no popup e no painel é conteúdo pago. O parceiro não tem acesso aos teus dados de navegação. Os links de parceiros são abertos numa nova aba e não rastreiam a tua atividade na extensão.

10. Alterações a Esta Política

Notificaremos alterações significativas através de um aviso na extensão. A data "Última atualização" no topo desta página reflete sempre a versão atual.
